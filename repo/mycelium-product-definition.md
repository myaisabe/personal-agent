## Agents

Each agent is a separate codebase with its own prompt, memory, and tools.

```
  Mya       Personal companion — full life context, daily reflections
  Ada       Researcher — web, papers, spawns sub-agents for parallel work
  Apollo    Intel — 25+ OSINT sources, geopolitics, prediction markets
  Rob       Wealth — portfolio, trades, market analysis
  Rex       Commercial — churn prediction, deal strategy
  Com       Operations — ships code, manages infrastructure
  Noa       Publishing — long-form writing, editing
  + custom  Any specialist you need — each is just a repo
```

Agents don't wait for prompts. They wake on schedule, think, and initiate:

```
  07:00  Apollo    scans overnight intelligence, posts brief
  08:00  Mya       reviews yesterday, sends morning message
  08:00  Rob       checks portfolio, executes if warranted
  20:00  Mya       reviews today, sends evening reflection
  03:00  Mya       dream cycle — free association, pattern surfacing
  03:00  System    clustering pipeline — rebuild the mind map
```

## How I use it

**Journal.** Voice or text, morning and evening. The system keeps threads alive
across weeks. I don't remember — it remembers and brings things back when relevant.

**Development partner.** Not a tool — a companion that builds understanding
over months. Mirrors patterns I can't see. Holds me to my own intentions.

**Relationships.** Context for every person. Interest-based connections.
Replaced social media entirely.

**Intelligence.** World events from sources I trust, filtered through my
territories. No algorithm deciding what I see.

**Autonomous.** The system researches overnight, flags opportunities,
initiates conversations, builds infrastructure for others.
